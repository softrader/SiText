# Future Features & Enhancements

## Planned Features

### File Management
- **Rename files**: Right-click context menu or Shift+Enter shortcut to rename files in place
- **Duplicate files**: Quick copy/duplicate option in context menu

### Editor Improvements
- **Smart cursor navigation**: When cursor is at the start of text on the bottom line, Down arrow should move cursor to END of that line (instead of no action)
- **Multi-cursor editing**: Cmd+D to select next occurrence, Cmd+Shift+L for all occurrences
- **Line numbers**: Optional toggle for line numbers in editor gutter
- **Word count**: Show word/character count in status bar
- **Auto-save indicator**: Visual feedback when file is being saved
- **Opening a file**: Can we save the cursor location in the file the last time it was open? and out it there when we access it next time?

### Search & Navigation
- **Fuzzy file search**: Cmd+P quick file switcher with fuzzy matching
- **Search within file**: Cmd+F for in-file search with highlight

### Linking & Backlinks
- **Backlinks panel**: Show which files link TO the current file

### Export & Integration
- **HTML export**: Static site generation from notes
- **Markdown export with assets**: Bundle images/attachments when exporting

### Mobile & Cross-Platform
- **iOS app**: Native iOS companion app with sync
  - Touch-optimized interface
  - Swipe gestures for navigation
  - Share extension to capture notes from other apps
  - Offline-first with sync when connected

### UI/UX Enhancements
- **Tabs**: Open multiple files in tabs
- **Dark mode improvements**: Enhanced contrast options

### Advanced Features
- **Git integration**: Commit, diff, and view history from within app
- **Daily notes**: Quick Cmd+D to create/open today's note
- **Checklist tracking**: Aggregate all `[ ]` checkboxes across files
- **Checklist Items Clickable** When you click them it completes them
- **Encryption**: Optional AES encryption for sensitive notes

### Performance
- **Lazy loading**: Load file contents on demand for large directories
- **Virtual scrolling**: Handle thousands of files efficiently
- **Background indexing**: Async content indexing with progress indicator

### Settings & Customization
- **Keyboard shortcuts editor**: Customize all shortcuts
- **Custom CSS**: User stylesheet for editor appearance in settings
- **Auto-tag extraction**: Parse frontmatter or inline tags

## Ideas to Explore

- **Image paste**: Paste images directly from clipboard into notes
- **Table editor**: WYSIWYG table editing mode
